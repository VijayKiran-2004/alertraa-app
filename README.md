# Alertraa - Health Monitoring Application

**Personal Health Guardian App**

This repository contains the Alertraa health monitoring application, converted to React Native for Android.

## 📱 Android App

The native Android app is located in the `AlertraaAndroid/` folder.

### Quick Start

```bash
cd AlertraaAndroid
npm install
npm run android
```

For detailed setup instructions, see [AlertraaAndroid/README.md](AlertraaAndroid/README.md)

## 🎯 Features

- 📊 Real-time health metrics tracking (Heart Rate, Blood Pressure, Blood Oxygen)
- 📅 Doctor appointment booking and management
- 💊 Medicine e-commerce with prescription management
- 👤 Comprehensive user profile with health history
- 🆘 Emergency SOS button with quick access
- 🌓 Dark mode support
- 📱 Native Android experience
